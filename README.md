# KRecall

# How to compile
## Use conveyor to compile the project.

### Output Test Dir

`./gradlew desktopJar`
For Windows:
`conveyor make windows-app`
For current OS:
`conveyor make app`
More information see: https://conveyor.hydraulic.dev/17.0/tutorial/hare/jvm/#__tabbed_2_2


### Output web
`./gradlew desktopJar`

`conveyor make site`
